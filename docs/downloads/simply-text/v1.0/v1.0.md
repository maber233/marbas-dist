# v1.0

## Summary
A lightweight macOS menu bar utility that strips formatting from clipboard text.

## New Features
- Automatic mode continuously monitors the clipboard and strips formatting instantly.
- Shortcut mode cleans clipboard text only when triggered via a configurable global hotkey.
- Menu bar icon with quick access to settings.
- About window with app info and version display.
- Custom app icon.
- Minimal footprint with background-agent behavior and no Dock icon.
