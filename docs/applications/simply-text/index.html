Simply Text

Simply Text is a lightweight macOS menu bar utility that strips formatting from copied text. No more fighting with unwanted fonts, colors, or styles when pasting into emails, documents, or chat.

Copy text from anywhere — browsers, PDFs, design tools, email — and Simply Text removes all formatting, leaving you with clean plain text ready to paste.
