# Simply Text

**Clean clipboard, pure text — every time.**

Simply Text is a lightweight macOS menu bar utility that strips formatting from copied text. No more fighting with unwanted fonts, colors, or styles when pasting into emails, documents, or chat.

## What It Does

Copy text from anywhere — browsers, PDFs, design tools, email — and Simply Text removes all formatting, leaving you with clean plain text ready to paste.

## Key Features

- **Automatic mode** — formatting is stripped the moment you copy.
- **Shortcut mode** — clean up on demand with a configurable global hotkey.
- **Menu bar-only** — runs quietly in the background with zero distractions.
- **Start at login** — set it once and forget about it.
- **Private by design** — clipboard data never leaves your Mac. No history, no network, no tracking.

## Requirements

macOS 13 Ventura or newer.
